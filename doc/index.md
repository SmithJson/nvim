## 安装

### Python

```
brew install python
```

### Neovim

```
brew install nevvim --HEAD
```

### luajit

```
brew install luajit --HEAD
```

### Ranger

```
pip3 install ranger-fm
pip3 install pynvim
```

### 安装配置

```
git clone https://github.com/SmithJson/nvim.git ~/.config/nvim
```

### 启动 nvim

```
# 自动下载依赖插件、下载完毕后重启
nvim
```
